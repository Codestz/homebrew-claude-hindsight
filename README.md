# Homebrew Tap — Claude Hindsight

Homebrew formulae for [Claude
Hindsight](https://github.com/Codestz/claude-hindsight), the
observability layer for your Claude Code sessions.

## Install

```bash
brew tap Codestz/claude-hindsight
brew install claude-hindsight

Upgrade

brew update
brew upgrade claude-hindsight

Uninstall

brew uninstall claude-hindsight
brew untap Codestz/claude-hindsight

What is Claude Hindsight?

A single-binary tool that parses Claude Code JSONL session files and
gives you full visibility — execution trees, tool call tracking, error
detection, subagent chains — through a web dashboard and terminal UI.
Zero dependencies, local-only, open source.

- GitHub: https://github.com/Codestz/claude-hindsight
- Docs: https://codestz.github.io/claude-hindsight/
